## Description

Rafflit은 블록체인 기술을 활용해 온라인 래플의 투명성 문제를 해결하는 플랫폼입니다.
Account Abstraction을 통해 블록체인 지식 없이도 누구나 쉽게 참여할 수 있으며, 스마트 컨트랙트로 당첨자 선정 과정의 공정성을 보장합니다.
직관적인 UI/UX로 래플 생성과 참여가 간편하며, 모든 과정을 투명하게 확인할 수 있습니다.

## How to Run

```sh
fnm use # or `nvm use`
corepack enable
pnpm install
pnpm client dev
```
